Обычная симуляция нотификаций в тг:

Добавить Task
add title.body.HH:MM:SS
Пример:
add "someTitle.someBody.10:22:33"
или:
add "someTitle.someBody.10:22:33 someTitle2.someBody2.11:23:44"

Удалить Task по ID
del id
Пример:
del 0
или:
del 1 2 3 4

Показать все Task-и:
list
